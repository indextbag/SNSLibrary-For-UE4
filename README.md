# SNSLibrary-For-UE4

It is UE4 plugin for SNS posting.  

## Support( v0.1 )
### Android
- Post with text and image to various SNS ( using intent )
- Reflect screenshot in gallery ( using mediascan )
### Windows( probably other platforms )
- Only Text

Please check the sample project
